Step 1. Install dependencies

  1. brew install aria2

  2. brew install google-cloud-sdk, configure with gcloud init

3. download Sen2Cor, then sh Sen2Cor-02.09.00-Darwin64.run --target ~/.sen2r/sen2cor290

  1. download Sen2Cor, then sh Sen2Cor-02.10.01-Darwin64.run --target ~/.sen2r/sen2cor (updated on 2022-02-09)

  2. brew install gdal -s
